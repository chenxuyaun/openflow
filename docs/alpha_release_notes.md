# OpenFlow Alpha Release Notes

## Product Position

OpenFlow Alpha is an AI collaboration workspace for long-running work that
should survive role changes, fresh sessions, and interrupted progress.

## What Alpha Includes

- Natural-language workspace bootstrap from `goal + initial_prompt`
- File-based project memory through materials, decisions, handoffs, and timeline
- Fresh-session work steps with completion, handoff, and review flow
- Research pack ingest, batch ingest, and linked decision support
- Default workspace language for ordinary users, with advanced workflow tools kept available

## Current Default Experience

The default workspace now emphasizes:

- current goal
- available materials
- current progress
- suggested next step

Advanced workflow, decision, and review controls remain available as secondary
workspace tools.

## What Alpha Does Not Try To Finish

Alpha does not include the broader Beta backlog such as deeper visual polish,
stronger first-run attraction, richer knowledge operations, or architecture
separation work.

## Beta Direction

The next planned Beta priority is a stronger first-run experience:

- more compelling landing storytelling
- better guided onboarding
- clearer value framing for non-technical users
